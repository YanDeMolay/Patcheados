---
title: "Como Usar o Sistema de Pesquisa"
---

O sistema de pesquisa funciona através de **parâmetros** e **termos**.
Cada termo representa uma característica do conteúdo pesquisado.

Você pode usar tanto o **nome principal do termo ou parâmetro** quanto seus **pseudônimos**.
Todos funcionam da mesma forma.

---

## Tags

**parâmetro:** `tag:`

Use este parâmetro para filtrar jogos por gênero, tema ou engine.

Ele também funciona na página **Círculos** para os jogos com patches trabalhados.

## Ordenação

**parâmetro:** `ordem:`

Use este parâmetro para definir a ordem dos resultados da pesquisa.

**pseudônimos:** `order:`

---

### z-a

Ordena os jogos/autores/círculos pelo nome em ordem alfabética decrescente.

---

### recente

Ordena os jogos pelo lançamento mais recente.

**pseudônimos:** `newest`

---

### antigo

Ordena os jogos pelo lançamento mais antigo.

**pseudônimos:** `oldest`

---

### patchrecente

Ordena os patches pelo lançamento mais recente.

**pseudônimos:** `patchnewest`

---

### patchantigo

Ordena os patches pelo lançamento mais antigo.

**pseudônimos:** `patcholdest`

## Remoção

Use um **" - "** antes de qualquer parâmetro para fazer uma pesquisa inversa

Por exemplo, `-plataforma:pc` irá remover da pesquisa todos os jogos lançados para PC.

## Filtros

Use esses parâmetros para filtrar informações específicas.

Cada parâmetro aqui funciona apenas com os termos (valores) definidos em sua respectiva lista.

---

**parâmetro:** `plataforma:`

Use este parâmetro para filtrar jogos por plataforma.

**pseudônimos:** `platform:`

**valores:** `plataformas`

---

**parâmetro:** `patchplataforma:`

Use este parâmetro para filtrar patches por plataforma.

**pseudônimos:** `patchplatform:`

**valores:** `plataformas`

---

**parâmetro:** `criador:`

Use este parâmetro para filtrar por autores/círculos do patch.

Na página **Autores**, ele irá filtrar por círculos.

Na página **Círculos**, ele irá filtrar por autores.

**pseudônimos:** `creator:`

---

**parâmetro:** `papel:`

Use este parâmetro na página **Autores** para filtrar pela função exercida em cada patch trabalhado.

**pseudônimos:** `role:`

**valores:** `papéis`

---

**parâmetro:** `personagem:`

Use este parâmetro na página **Autores** para filtrar pelo nome dos personagens interpretados.

**pseudônimos:** `character:`

---

**parâmetro:** `idioma:`

Use este parâmetro para filtrar pelo idioma original do jogo.

**pseudônimos:** `language:`, `lingua:`, `língua:`

**valores:** `idiomas`

---

**parâmetro:** `origem:`

Use este parâmetro para filtrar pelo idioma de origem do conteúdo traduzido no patch.

**pseudônimos:** `origin:`

**valores:** `idiomas`

---

**parâmetro:** `região:`

Use este parâmetro para filtrar pelas regiões do jogo.

**pseudônimos:** `region:`, `regiao:`

**valores:** `regiões`

---

**parâmetro:** `patchregião:`

Use este parâmetro para filtrar pelas regiões do patch.

**pseudônimos:** `patchregion:`, `patchregiao`

**valores:** `regiões`

---

**parâmetro:** `status:`

Use este parâmetro para filtrar pelo status do patch.

**valores:** `status`

---

**parâmetro:** `desenvolvedora:`

Use este parâmetro para filtrar pela desenvolvedora do jogo.

**pseudônimos:** `developer:`, `dev`

---

**parâmetro:** `distribuidora:`

Use este parâmetro para filtrar pela distribuidora do jogo.

**pseudônimos:** `publisher:`, `pub`

---

**parâmetro:** `patchdistribuidora:`

Use este parâmetro para filtrar pela distribuidora do patch.

**pseudônimos:** `patchpublisher:`, `patchpub`

---

**parâmetro:** `lançamento:`

Use este parâmetro para filtrar pelo ano de lançamento do jogo.

**pseudônimos:** `release:`, `lancamento`

---

**parâmetro:** `patchlançamento:`

Use este parâmetro para filtrar pelo ano de lançamento mais recente do patch por plataforma disponível.

Caso combinado com `patchplataforma:` o ano será especifico para cada plataforma.

**pseudônimos:** `patchrelease:`, `patchlancamento`

---

**parâmetro:** `jogadores:`

Use este parâmetro para filtrar por máximo de jogadores.

Vocẽ pode usar um **" - "** entro os números para pesquisar um intervalo.

Por exemplo, `jogadores:2-4` irá mostrar jogos para 2, 3 e 4 jogadores.

**pseudônimos:** `players:`

---

**parâmetro:** `nsfw:`

Use este parâmetro para filtrar por jogos com conteúdo NSFW.

**valores:** `estado`

---

**parâmetro:** `lost_source:`

Use este parâmetro para filtrar por patches com distribuição original perdida.

**valores:** `estado`

---

**parâmetro:** `lost_media:`

Use este parâmetro para filtrar por patches considerados lost media.

**valores:** `estado`

---

**parâmetro:** `mtl:`

Use este parâmetro para filtrar por patches traduzidos por máquina.

**pseudônimos:** `machine_translation:`

**valores:** `estado`

---

**parâmetro:** `dub_ia:`

Use este parâmetro para filtrar por patches com dublagem por inteligência artificial.

**pseudônimos:** `ai_dub:`

**valores:** `estado`

---

**parâmetro:** `externo:`

Use este parâmetro para filtrar por patches disponibilizados de forma externa ao jogo.

**pseudônimos:** `external:`

**valores:** `estado`

---

**parâmetro:** `legendas:`

Use este parâmetro para filtrar pela disponibilidade das legendas.

**pseudônimos:** `subs:`

**valores:** `estado`

---

**parâmetro:** `gráficos:`

Use este parâmetro para filtrar pela disponibilidade dos gráficos.

**pseudônimos:** `graphics:`, `graficos`

**valores:** `estado`

---

**parâmetro:** `dublagem:`

Use este parâmetro para filtrar pela disponibilidade da dublagem.

**pseudônimos:** `dub:`

**valores:** `estado`

---

**parâmetro:** `oficial:`

Use este parâmetro para filtrar por jogos com patches oficiais.

**pseudônimos:** `official:`

**valores:** `estado`

---

**parâmetro:** `reupload:`

Use este parâmetro para filtrar por patches com reupload não oficial.

**pseudônimos:** `archive:`

**valores:** `estado`
